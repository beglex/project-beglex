## Description

Backend part based on [NestJS](https://nestjs.com/) framework.






## Project setup

Install dependencies:
```bash
  npm i
```

Prepare configuration:
```bash
  cp .env.sample .env
```






## Compile and run the project

```bash
  # Development
  npm run start:dev

  # Production
  npm run start:prod
```






## Run tests

```bash
  # Unit and E2E tests
  npm run test

  # Coverage
  npm run test:cov
```
