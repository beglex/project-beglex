## Description

Frontend part based on [Next.js](https://nextjs.org/) framework.






## Project setup

Install dependencies:
```bash
  npm i
```

Prepare configuration:
```bash
  cp .env.sample .env
```






## Run the project

```bash
  # Development
  npm run dev

  # Production
  npm run build
```
